## Gropuie tracker

**Groupie Tracker is a project for kood/JÃµhvi. In this project we are handling remote JSON data, unmarshaling it and then serving it for the user via HTTP server. The server is generated and maintained by net/http and html/template standard GO packages.**

### Setup
Clone the repository
```
git clone https://01.kood.tech/git/abuschma/groupie-tracker
```

### Usage
1. Run server from terminal:
```
go run .
```
2. Open [localhost:8080](http://localhost:8080/)
3. Enjoy!
